# Scott Rallya public portfolio

A source-led portfolio of bounded engineering proofs, research-method artifacts, and a deliberately small public orientation for a proposed platform.

## Structure

| Directory | Purpose | Maturity |
| --- | --- | --- |
| [01-knowledge-governance](01-knowledge-governance/) | Controlled vocabulary and portable validation design for a file-primary knowledge system. | Pilot taxonomy and rule artifacts. |
| [02-bounded-retrieval](02-bounded-retrieval/) | Runnable, local-only ContextPack retrieval proof with provenance boundaries. | Working proof, not a product runtime or semantic retrieval system. |
| [03-research-methods](03-research-methods/) | Public-safe research-method material, coupling criteria, and curated technical reflections. | Draft methodology and individual observations. |
| [04-ml-boundaries-and-creative-experiments](04-ml-boundaries-and-creative-experiments/) | Explicit boundary between current retrieval work, future ML/RL learning, and a small creative-model artifact. | Learning and design record, not ML research results. |
| [05-exocore-platform](05-exocore-platform/) | Public-safe Exocore orientation and first-proof boundary. | Proposed direction, not implemented product. |

## Architectural throughline

The portfolio is deliberately organized as a chain of accountable surfaces:

```text
knowledge governance
  -> bounded retrieval with provenance
  -> research protocol and human review
  -> explicit ML learning boundaries
  -> local-first Exocore platform proposal
```

The root [source manifest](PORTFOLIO-SOURCE-MANIFEST.md) records copied source artifacts, public-safe derivatives, and files intentionally withheld from publication.

## Status and publication boundary

This repository is an artifact-led public portfolio. It contains no private session transcripts, raw personal disclosures, credentials, or runtime configuration. Each artifact labels its maturity and source treatment. Read [PUBLISHING-GATE.md](PUBLISHING-GATE.md) for the release boundary and [LICENSE](LICENSE) for reuse terms.
