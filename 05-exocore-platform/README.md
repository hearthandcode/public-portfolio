# 05. Exocore platform and return path

## What this directory shows

This directory contains a deliberately small public orientation for Exocore, a proposed local-first workbench for inspectable knowledge work, bounded delegation, and reliable return to prior work.

## Artifacts

- `artifacts/exocore-public-orientation.md`: the public-safe platform direction, current public evidence, and non-claims.
- `artifacts/exocore-first-proof-boundary.md`: the proposed first-proof posture, its boundaries, and its evidence discipline.

## Role in the wider architecture

Exocore is the intended integration surface for the prior directories: governed records, bounded retrieval, visible review, and human authority. The public materials preserve only the architectural commitments needed to understand that direction. Internal implementation options, dependency research, review prompts, and operational design remain private.

## Boundary

These are review drafts and proposed direction. They do not establish a launched desktop system, autonomous agent runtime, broad user adoption, or completed product architecture. The documentation keeps proposed elements separate from public code and future proof results.
