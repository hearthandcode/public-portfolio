# Artifact inventory

- `knowledge-taxonomy.yaml` is a public-safe pilot controlled-vocabulary registry. It defines term schemes, lifecycle states, evidence states, privacy tiers, and related metadata boundaries.
- `taxonomy-integrity-rules.yaml` is a public-safe companion rule set. It describes selected machine-readable constraints, including uniqueness, reference validity, membership, and graph structure.
- `ai-ready-knowledge-architecture.md` is a public-safe design record that connects controlled metadata, provenance, canonical records, and human review to the needs of AI-ready context.

All three files are concise public-safe derivatives. They are presented together because the taxonomy names the governance vocabulary, the rules make parts of that vocabulary checkable in principle, and the brief explains the broader information-architecture rationale. They do not expose private source locations, personal records, employer-specific application context, or runtime implementation details.
